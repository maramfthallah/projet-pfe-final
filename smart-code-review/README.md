# Smart Code Review

A modern code review assistant built with **React 19** and **Vite**.

## Getting Started

```bash
npm install
npm run dev
```

## Scripts

| Command           | Description                  |
| ----------------- | ---------------------------- |
| `npm run dev`     | Start dev server (port 3000) |
| `npm run build`   | Production build             |
| `npm run preview` | Preview production build     |
